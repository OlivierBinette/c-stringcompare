Detailed Description

Longest Common Substring (LCS) distance.

Construct a new LCSDistance object.

The LCS distance is the sum of the lengths of the string minus twice the length of their longest common substring.

By default, the LCS distance dist is normalized to 2 * dist / (len + dist).

The (unnormalized) similarity score is defined as the length of the longest common substring.

The normalized similarity score is 1 minus the normalized distance.

Parameters
normalizeWhether to normalize the distance/similarity to be between 0 and 1. Defaults to true.
similarityWhether to return a similarity score rather than a distance. Defaults to false.
dmat_sizeDefault starting string buffer size. If the maximum string length s_max is known in advance, this can be set to s_max + 1 to improve efficiency.
