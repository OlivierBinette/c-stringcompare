Detailed Description

template<class dtype>
class stringcompare::Comparator< dtype >

Base class for comparators.

Requires a compare() function. Implements the callable operator()(), and the elementwise() and pairwise() functions.

Template Parameters
dtypeType of objects to compare (typically string).
