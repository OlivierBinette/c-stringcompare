Detailed Description

Levenshtein distance.

This is the minimal number number of deletions, insertions, and substitutions needed to transform one string into the other.

Construct a new Levenshtein object.

By default, the Levenshtein distance dist is normalized to 2 * dist / (len + dist).

For two strings and , with (unnormalized) Damerau-Levenshtein distance (minimal number of deletions, insertions, and substitutions needed to transform one string into the other), the similarity score is defined as .

The normalized similarity score is defined as 1 minus the normalized distance.

Parameters
normalizeWhether to normalize the distance/similarity to be between 0 and 1. Defaults to true.

dmat_sizeDefault starting string buffer size. If the maximum string length s_max is known in advance, this can be set to s_max + 1 to improve efficiency.
