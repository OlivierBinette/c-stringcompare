Construct a new \mbox{\hyperlink{classstringcompare_1_1Levenshtein}{Levenshtein}} object. 

By default, the \mbox{\hyperlink{classstringcompare_1_1Levenshtein}{Levenshtein}} distance {\ttfamily dist} is normalized to {\ttfamily 2 $\ast$ dist / (len + dist)}.

For two strings $ s $ and $ t $, with (unnormalized) Damerau-\/\+Levenshtein distance $ \texttt{dist} $ (minimal number of deletions, insertions, and substitutions needed to transform one string into the other), the similarity score is defined as $ \texttt{sim} = \s\+\t\ - \texttt{dist} $.

The normalized similarity score is defined as 1 minus the normalized distance.
