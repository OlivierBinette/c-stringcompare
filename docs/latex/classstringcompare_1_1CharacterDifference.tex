Construct a new \mbox{\hyperlink{classstringcompare_1_1CharacterDifference}{Character\+Difference}} object. 

Calls to \mbox{\hyperlink{classstringcompare_1_1Comparator_a1d9536f8a02c3d1a936879aba116ee00}{operator()()}} and \mbox{\hyperlink{classstringcompare_1_1CharacterDifference_a4821c21e1339d65dcfd869617e3a1d55}{compare()}} return the comparison value.

Given two strings, let $ A $ and $ B $ be their character multisets. Then the \mbox{\hyperlink{classstringcompare_1_1CharacterDifference}{Character\+Difference}} distance between the two strings is defined as $ d = |A| + |B| - 2|A \cap B| $.

By default, the distance is normalized to $ 2d / (|A| + |B| + d) $.

The similarity score is defined as $ |A \cap B| $ and its normalization is \[ \frac{|A \cap B|}{|A| + |B| - |A \cap B|}. \]
