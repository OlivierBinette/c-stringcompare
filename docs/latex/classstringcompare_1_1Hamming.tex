\doxysubsection{Detailed Description}
\mbox{\hyperlink{classstringcompare_1_1Hamming}{Hamming}} distance between two strings. 

The \mbox{\hyperlink{classstringcompare_1_1Hamming}{Hamming}} distance is the number of non-\/matching characters when comparing the strings by corresponding positions.

Construct a new \mbox{\hyperlink{classstringcompare_1_1Hamming}{Hamming}} object. 

By default, the \mbox{\hyperlink{classstringcompare_1_1Hamming}{Hamming}} distance {\ttfamily dist} is normalized to {\ttfamily dist / len}, where {\ttfamily len} is the length of the longest string.

The (unormalized) similarity score is defined as the length of the longest string minus the \mbox{\hyperlink{classstringcompare_1_1Hamming}{Hamming}} distance.

The normalized similarity score is defined as 1 minus the normalized distance.
