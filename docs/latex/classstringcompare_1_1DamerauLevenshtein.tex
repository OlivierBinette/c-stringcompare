\doxysubsection{Detailed Description}
Compute the Damerau-\/\+Levenshtein distance. 

This is the number of deletions, insertions, substitutions and transpositions needed to transform one string into the other.

Construct a new \mbox{\hyperlink{classstringcompare_1_1DamerauLevenshtein}{Damerau\+Levenshtein}} object. 

By default, the Damerau-\/\+Levenshtein distance {\ttfamily dist} is normalized to {\ttfamily 2 $\ast$ dist / (len + dist)}.

For two strings $ s $ and $ t $, with (unnormalized) Damerau-\/\+Levenshtein distance $ \texttt{dist} $ (minimal number number of deletions, insertions, substitutions and transpositions needed to transform one string into the other), the similarity score is defined as $ \texttt{sim} = \s\+\t\ - \texttt{dist} $.

The normalized similarity score is defined as 1 minus the normalized distance.
